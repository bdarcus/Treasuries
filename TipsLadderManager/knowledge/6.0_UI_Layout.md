# 6.0 UI Layout Specification

## Dependencies
**Requires:** 5.0 UI Schema

**This file specifies the authoritative visual standards, layout structure, and responsive behavior for the TIPS Ladder Manager.**

---

## Visual Theme: Teal High-Contrast

### Color Palette
| Element | Color | Value |
|---|---|---|
| Page Background | Light Gray | `#fafafa` |
| Card Background | Deep Teal | `#1B4242` |
| Card Border | Muted Teal | `#2D5A5A` |
| Label Text | Mint Teal | `#9CCBCB` |
| Focus Label / Checkbox | White | `#fff` |
| Input Background | Obsidian Teal | `#092626` |
| Primary Button | Sage Teal | `#5BA4A4` |
| Button Hover | Lighter Sage | `#76B9B9` |
| Error Text | Soft Red | `#f87171` |

---

## Top Card Geometry

The `.form-card` must remain perfectly stable during mode switches and after execution.

- **Static Width:** `1100px`
- **Static Height:** `160px`
- **Positioning:** Relative (container for absolute sub-elements)
- **Shadow:** `0 10px 15px -3px rgb(0 0 0 / 0.4)`

---

## Header & Information Quadrant

The top of the card is divided into a **Left Action Block** and a **Right Info Stack**.

### Left Action Block
- **Mode Toggles:** Rebalance / Build buttons.
- **Run Button:** "Run Rebalance" or "Build Ladder".
- Gap: `16px` between toggles and Run button.

### Right Info Stack (`.info-stack`)
- **Row 1:** FedInvest Price Date + Ref CPI Date.
- **Row 2:** TIPS Count + Ref/Mkt links + Export buttons + Duration links.
- **Separators:** Middle dot (` · `) between inline items.

---

## Form Field Arrangement

- **Layout:** Flexbox with `gap: 20px`.
- **Vertical Alignment:** `flex-end` (aligns inputs to bottom).
- **Labels:** Uppercase, `800` weight, `10px` font size.
- **Pre-ladder Interest:** Label color must be `#fff` for maximum visibility.

---

## Output Table Alignment

To maintain visual symmetry, the data output must align with the top card.

- **Table Wrap:** `min-width: 1100px` (matches card width).
- **Behavior:**
    - In **Rebalance** mode, the table naturally fills or exceeds `1100px`.
    - In **Build** mode, the table may be narrower, but its container must remain `1100px` wide.
- **Max Height:** Calculated dynamically to ensure the header remains sticky without overlapping page content.

---

## Status Bar & Net Cash

- **Status Message:** Displayed below fields, white text, `12px`.
- **Net Cash / Total Cost:** Absolutely positioned at the bottom right of the card to prevent layout shift.
    - `bottom: 44px`, `right: 18px`.
    - Label: Mint Teal (`#9CCBCB`).
    - Value: Large White (`18px`, `#fff`).
